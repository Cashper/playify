COMMANDS

/play [query]

Plays a song, playlist, or searches by title from YouTube, Spotify, or SoundCloud.

/pause

Pauses the current song.

/resume

Resumes the paused song.

/skip

Skips to the next song in the queue.

/loop

Toggles looping for the current song.

/stop

Stops playback and disconnects the bot.

/kaomoji

Toggles kawaii mode for fun responses (admin only).

/language

Sets the bot's language (English/French) for the server (admin only).
